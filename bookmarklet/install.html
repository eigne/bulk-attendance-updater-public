勤務一括編集 <-- ブックマークバーにドラッグ＆ドロップ

使い方

  1. 一括編集を開く
  2. 一括編集画面でブックマークをクリックする
  3. 出社日を選択する（例：水木出社）

    4. ※休日を選択しない。選択したい場合、休日スキップをオフにして

  4. 勤務パターンを選択し、変更ボタンを押すと反映される
  5. 反対ボタンを押し、残りの勤務パターンをセットする（例：在宅)

    6. ※反対しても休日を選択しない。選択したい場合、休日スキップをオフにして

  6. 保存ボタンを押すとおしまい

    7. ※休日に勤務パターンを入れても特に問題ないけど見づらくなる。
